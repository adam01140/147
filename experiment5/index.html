Description

In this programming assignment, I built a generator that directly reads 1 of 3 inspiring images and then uses that image to render parameterized images


It then adjusts its own designs to optimize an evaluation score and find the image that best fits our inspiration.

Technical

  1. This project integrates inspiring images that are selectable from the drop-down on the display page.
  2. Depending on what inspiring image is selected, the program uses different types of shapes.
  3. The prgram also also varies the amount of shapes used based on each image .
  4. The design is rendered in a way that is meant to mimic the inspiring images.
  5. The mutation process makes incremental changes to the inital design.
  6. The rate parameter should is implemented in a way that allows the user to almost stop exploration by sliding the rate down to 1%.
  7. The generator responds accuratly to the mutation rate slider.

Annealing

The most interesting results from this kind of evolutionary generator are often achieved by continually adjusting the mutation rate. Set the slider to 100%, reset the generator, and then slowly drag the slider down to 1% over the period of about 30 seconds. This allows the generator to make large design changes at the start while allowing it to focus on finely details towards the end. This is best done by hand while watching the outputs improve because you can use your human judgment to decide when the generator needs to be making bigger or smaller changes to keep making progress.
